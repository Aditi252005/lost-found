Lost&Found is a campus portal to report and recover lost or found items.

 • Built a full-stack web application to help students report, search, and manage lost and found items across campus, with secure
 user authentication and a personalized "My Reports" dashboard.
 • Integrated Cloudinary for efficient image upload, storage, and delivery of item photos, handled backend via Restful APIs and
 dynamic MongoDB models.
 • Tech Stack : Node.js, Express, MongoDB, Cloudinary

 
Link:- https://lost-found-g4me.onrender.com
